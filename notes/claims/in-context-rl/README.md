# In-Context RL Claim Ledger

This directory stores reusable, note-grounded claims extracted from `notes/papers/*.md`.

## Files

- `foundation.yaml`: claims about the existence, mechanism, and conceptual framing of ICRL
- `methodology.yaml`: claims about objectives, data design, memory, and architectural choices
- `evaluation.yaml`: claims about benchmarks, metrics, and failure regimes
- `systems_and_data.yaml`: claims about scaling, interfaces, long context, and supervision dependencies
- `contradictions.yaml`: explicitly conflicting or tensioned claims
- `weak_claims.yaml`: reusable claims with limited support or noisy source quality

## Interpretation

- `evidence`: directly supported by local notes and anchors
- `inference`: supported by multiple local notes but involves synthesis beyond a single note
- `speculation`: potentially useful, but not strong enough for downstream proposal use without further checking
